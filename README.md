[![](https://img.shields.io/badge/Download-Latest_Release-blue.svg)](https://github.com/pugnoseindiana362/wsdot-mcp-server/releases)

This application connects your computer to Washington State Department of Transportation data. It provides real-time updates on highway conditions, ferry schedules, and traffic alerts. You use this tool to gather information for planning your travel across the state. It works directly on your Windows computer.

## 📋 What this tool does

The wsdot-mcp-server gathers live data feeds from official Washington state sources. You can use it to check specific details without visiting multiple websites.

Key features:
* View current highway conditions and closures.
* Check ferry schedules for all major routes.
* Track ferry vessel locations on a map.
* Calculate toll rates for your planned trips.
* Review wait times at international border crossings.
* Receive automatic alerts for major road incidents.

This software acts as a bridge between official traffic data and your private computer. It organizes complex information into a format meant for easy reading.

## 🔍 Understanding the data

This tool pulls information directly from public records. You might see specific technical terms on your screen. Here is what those terms mean:

* MCP: This stands for Model Context Protocol. It helps the software share data with other programs you use.
* STDIO: This is a way the computer communicates internally to send data logs.
* WSDOT: This refers to the Washington State Department of Transportation.
* Vessel tracking: This displays the GPS coordinates of ferry ships currently in transit.
* Border waits: These values represent the time in minutes it takes to cross the international boundary.

## ❓ Common questions

**Does this app track my location?**
No. The software only downloads status reports for the zones you select. It does not record or send your personal location data to any server.

**How do I update the software?**
The application checks for new versions whenever you start it. If an update exists, a notification appears on the screen. Click the link in that notification to install the latest version.

**What should I do if the schedule data looks old?**
Click the Refresh button located in the toolbar. This forces the application to grab fresh data from the WSDOT servers.

**Can I run this on a Mac?**
This version is designed specifically for Windows.
